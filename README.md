# Justifica_edu
### Projeto de criação tecnológica para justificativa de falta de estudantes
O **JustificaEdu** é uma plataforma web desenvolvida para automatizar e otimizar o gerenciamento de justificativas de faltas em escolas públicas. O sistema centraliza o recebimento de atestados e documentos, direcionando as informações de forma automática e personalizada apenas para os professores responsáveis por cada turma, eliminando processos manuais e planilhas burocráticas.

### Objetivos de Desenvolvimento Sustentável (ODS)

Este projeto está diretamente alinhado com as metas da ONU:

* **ODS 4 – Educação de Qualidade:** Torna a gestão escolar mais eficiente, permitindo que os professores foquem no ensino e no acompanhamento pedagógico dos estudantes.
* **ODS 3 – Saúde e Bem-Estar:** Reduz a sobrecarga de trabalho administrativo e o estresse dos profissionais da educação, além de organizar o acompanhamento de ausências motivadas por questões de saúde dos alunos.
